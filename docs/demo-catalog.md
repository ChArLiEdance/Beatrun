# Beatrun Demo Catalog

This catalog documents the audio metadata used by the current Beatrun competition MVP.

Important limits:

- Audio playback is generated locally by Beatrun.
- No unauthorized commercial music is downloaded, scraped, streamed, bundled, or redistributed.
- Matching is 1:1 only.
- Tempo adjustment must stay within +/-10%.
- Double-time and half-time matching are disabled.

## Shared Audio Rights

All current MVP tracks use the same safe demo rights profile.

| Field | Value |
| --- | --- |
| Rights status | Generated in app |
| License/source type | Original generated demo audio |
| Attribution | Generated by Beatrun's local AVFoundation synthesis engine |
| Source link | `Beatrun/MetronomeEngine.swift` |
| Source explanation | Track metadata drives a local drum/bass loop; there is no external recording |
| Tempo adjustment | Allowed |
| Redistribution risk | No third-party recording is redistributed |

## Matching Evidence

The app calculates:

- `speedRatio = targetCadence / originalBPM`
- `speedChangePercent = (speedRatio - 1) * 100`
- reject when `abs(speedChangePercent) > 10`
- reject when rights metadata does not allow tempo adjustment

Because adjusted playback is generated at the selected target cadence, adjusted BPM equals the target cadence for accepted tracks. The UI shows original BPM, adjusted BPM, speed change, match score, confidence, and rights status.
