# Beatrun

Beatrun is an iOS app built with SwiftUI.

## Requirements

- Xcode 16 or later
- iOS 18 or later

## Getting Started

Open `Beatrun.xcodeproj` in Xcode, select an iPhone simulator, then run the `Beatrun` scheme.
